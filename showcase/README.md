# AI Video + 3D + Poster Showcase

This project uses `assets/hero.png` as the base image.

## Output locations (drop files here)

Put generated files in `assets/generated/` with these exact names:

- `poster_massive_nature_1920x1080.png` (Image poster with massive epic-nature background)
- `3d_view_front.png`
- `3d_view_angle.png`
- `3d_view_close.png`
- `ai_video.mp4` or `ai_video.webm` or `ai_video.mov` (optional: exported final video file)

Prompts live in `showcase/`:

- `showcase/ai_video_prompt.txt` (15s video prompt + storyboard)
- `showcase/3d_design_prompt.txt` (3D renders prompt + Blender notes)

Once present, open `showcase.html` to view them one by one.
